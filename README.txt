# Шахматная комбинаторная задача

Описание:
Решение задачи расстановки 5 различных шахматных фигур (Король, Ферзь, Ладья, Слон, Конь) на доске так, чтобы ни одна фигура не била другую.

Задача:
Найти все возможные расстановки 5 фигур:
- Король (K)
- Ферзь (Q) 
- Ладья (R)
- Слон (B)
- Конь (N)

Условие:
где ни одна фигура не находится под боем другой.

Технологии:
- Python 3.8+
- Алгоритмы комбинаторики

- Верификация шахматных правил
